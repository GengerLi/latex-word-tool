LaTeX 公式转 Word 可编辑公式

输入 LaTeX 或上传公式图片，一键生成 Word 中可编辑的数学公式。

所有逻辑都在浏览器中运行；导出的 Word 文件仅包含居中的 OMML 数学公式，不包含标题、源码、说明文字或图片。

使用提示

          

          
    
            
  • 导出前建议先查看 KaTeX 预览，确认公式主体无明显语法问题。
    • 
            
  • 浏览器直接调用千问接口可能受网络、CORS 或浏览器安全策略影响；失败时请检查 API Key、网络状态和图片清晰度。
    • 
            
  • 复杂矩阵、对齐环境会尽量降级为 Word 公式区域中的可编辑文本，确保文件可以打开且内容不丢失。
